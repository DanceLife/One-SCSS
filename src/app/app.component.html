What are you getting

With this boilerplate you are acquiring the basic structure to star implementing your project using SASS3, the newer version of SASS standing for Syntactically Awesome Style Sheets. This new version is known as SCSS for Sassy CSS; which is nothing but a superset of CSS3â€™s syntax allowing you to utilize your old CSS syntax while you move into the new syntax and features offered by SCSS. This project was started with Angular 6 and Bootstrap 4.

How to use it

As a quick starter you can go into src/scss/_variables.scss on the project and quickly set your color variables as desired. Then take a look at the app.component.scss where you will see the directive for importing the custom scss file, as well as a css rule utilizing a variable defined all the way back into the _variable.scss file, which modifies the original bootstrap color.

You can also add more variables by copy them from the node_modules\bootstrap\scss\_variables.scss file.
