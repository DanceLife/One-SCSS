What are you getting

With this boilerplate you are acquiring the basic structure to star implementing your project using SASS3/Bootstrap. SCSS is the newer version of SASS standing for Syntactically Awesome Style Sheets. This new version is known as SCSS for Sassy CSS; which is nothing but a superset of CSS3â€™s syntax allowing you to utilize your old CSS syntax while you move into the new syntax and features offered by SCSS.

How to use it

As a quick starter you can go into src/scss/_variables.scss and quickly set your color variables as desired.

On the app.component.scss you can see the directive for importing the custom scss file, as well as a rule using a variable defined all the way back into the _variable.scss file, modifying the original bootstrap color.

You can also add more variables by copy them from the node_modules\bootstrap\scss\_variables.scss file.

When creating a new .scss file in your project then add the following directive at the top of the file: @import '~src/scss/styles.scss';
